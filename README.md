# Documents API

API REST desenvolvida em Python utilizando FastAPI e arquitetura serverless na AWS.

## 🚀 Tecnologias
- Python 3.11
- FastAPI
- AWS Lambda
- API Gateway
- DynamoDB
- S3

## 📌 Objetivo
Fornecer uma API escalável para criação, versionamento e gerenciamento de documentos,
com autenticação, auditoria e armazenamento seguro.

## ▶️ Executar localmente
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
Acesse: http://localhost:8000/docs
